MINECRAFT BST

SERVER IP : BST.kgtmc.xyz

마인크레프트 BST 서버에 오신것을 환영합니다.
BST서버는 BJ, 스트리머, 유튜버 시청자들이 서버에 들어오 생야생으로 재료를 구한 뒤, 원하시는 BJ, 스트리머, 유튜버의 동상을 만드시거나 하시면 됩니다.
단, 서버에 규칙이 존재하니, 규칙을 필독하시고, 서버오 관련된 중요한 사항은 공식 페이스북에 올려 드립니다.

RULE

안내.

규칙을 대충 읽으신 후, 어기신 부분은 저희 관리자가 책임을 지지 않습니다. 또한 서버 내에 분쟁이 생기면 싸운 두 분이 밴이 됩니다.

1. 태러 관련.

서버를 태러 하시는 유저는 서버에서 바로 밴 예정입니다.

2. 건축물

서버 안에서의 건축은 자유롭습니다. 하지만, 건축물을 부수거나, 하실 경우엔 즉시 밴 처리 하겠습니다.

하지만, 불건전한 건축물이 보일 경우 관리자가 판단 하에 건축물 삭제 조치를 합니다.

어떤 BJ, 스트리머, 유튜버분의 요청이 있을 경우에도 삭제를 하겠습니다.

삭제가 필요한 건축물이 있다면 관리자의 디스코드로 연락 부탁 드립니다.

디스코드 : https://geuntae.kr/discord 에 오신후, 근태#3254로 디엠 부탁 드립니다.

3. 채팅

서버에서 다른사람에게 욕설, 성희롱 등을 하실 경우 서버에서 바로 밴이 됩니다.

BJ, 스트리머, 유튜버 분에게 욕설, 성희롱 발언도 밴 대상 입니다.
